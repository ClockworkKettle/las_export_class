# las_export_class

Used to export individual classes from a classified las-file

## Dependencies:

###### PDAL (Point Data And Abstraction Library)

###### GDAL (Geospatial Data Abstraction Library)

## Usage:


Classes:  
        1: Cars  
        2: Ground  
        3: Grass/low vegetation  
        4: Cars/Fences/Low walls  
        5: Trees/Tall hedges  
        6: Buildings  
        7: Noise
